# LearnVest MVP
## Gamified Financial Literacy Platform for India
### "Learn Money. Play Smart. Build Wealth." 🦊

---

## HOW TO RUN

### Windows
1. Double-click `run_windows.bat`
2. Browser opens at http://localhost:5000 automatically
3. Done!

### Mac / Linux
1. Open Terminal in this folder
2. Run: `chmod +x run_mac_linux.sh && ./run_mac_linux.sh`
3. Browser opens at http://localhost:5000 automatically

### Manual (any OS)
```
pip install flask
python app.py
```
Then open http://localhost:5000 in your browser.

---

## REQUIREMENTS
- Python 3.8 or higher
- pip (comes with Python)
- Any modern browser (Chrome recommended)
- No internet required after install

## FEATURES IN THIS MVP

### 🎓 Learning (Duolingo-style)
- 8 lessons covering: Money Basics, Saving, Budgeting, Banking, Stocks, Credit, Investing, Taxes
- Swipeable lesson cards with Indian examples
- 3-question quiz after each lesson with instant feedback
- XP + VestCoins rewards for correct answers
- Lesson path with locked/unlocked/completed states

### 🦊 Animated Fox Mascot
- 6 animations: idle bob, wave, celebrate (on correct answer), sad (on wrong), teach, level-up spin
- Expressive eyes change with each animation
- Rotating tip bubbles with financial advice
- Tail wags during idle, stock chart pulses during teaching

### 🌍 VestWorld (Minecraft-style)
- **Dalal Street**: Simulated Indian stock market with 6 stocks
  - Prices fluctuate randomly in real-time
  - Live sparkline charts for each stock
  - Buy/Sell with VestCoins
  - Portfolio P&L tracking
- **VestBank**: Save VestCoins, earn 5% weekly interest
- **My Business**: Start chai stall / repair shop / online store / tech startup
  - Collect weekly profits, business grows over time
- **Real Estate**: Buy studio / 1BHK / 2BHK / villa
- **Life Events**: Random financial challenges (market crash, Diwali bonus, medical emergency)

### 📊 Portfolio & Net Worth
- Total net worth tracker (Cash + Savings + Stocks + Property)
- Allocation breakdown with VestCoins breakdown
- Holdings with P&L per stock

### 🏆 Leaderboard & Badges
- Weekly XP leaderboard with 5 simulated competitors
- 8 achievement badges (First Lesson, 7-Day Streak, First Investment, etc.)

### 👤 Profile
- Custom name and avatar
- XP / Lessons / Streak stats
- Mock certificate download
- Game reset option
